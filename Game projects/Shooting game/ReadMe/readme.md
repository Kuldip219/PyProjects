# 🚀 X Hunter

A fast-paced 2D space shooting game built with **Python** and **Pygame**.

Take control of your spaceship, destroy enemy ships, survive as long as possible, and achieve the highest score!

---

## 🎮 Features

- 🚀 Smooth spaceship movement
- 🔫 Bullet shooting system
- 👾 Multiple enemy ships
- 💥 Explosion animations
- ❤️ Custom health bar
- 📊 Score system
- ⏸️ Pause menu
- 🏠 Main menu
- ⚙️ Options menu
- 💀 Game Over screen
- 🎨 Pixel-art graphics
- 🖱️ Interactive menu buttons
- 📱 Planned Android support
- 🌐 Planned browser version

## ▶️ Installation

Clone the repository:

```bash
git clone https://github.com/YOUR_USERNAME/X-Hunter.git
```

Go into the project folder:

```bash
cd X-Hunter
```

Install Pygame:

```bash
pip install pygame
```

Run the game:

```bash
python main.py
```

| Key | Action |
